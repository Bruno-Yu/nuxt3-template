# Nuxt3 + TypeScript + Tailwind + Jest + Pinia + element-plus starter

> Nuxt3 開發模板 ( Nuxt3 + TypeScript + Tailwind + Pinia + i8n + element-plus + dayjs ) Node18

## 使用技術

1. Nuxt3 (v3.11.2)
2. Node18
3. Typescript
4. Pinia / pinia-plugin-persistedstate
5. Tailwind
6. element-plus
7. i18n
8. dayjs

## 使用 lint 風格
1. Prettier


## 安裝指令

```bash
yarn install
```

## 執行開發環境

```bash
yarn dev
```

## 建構

```bash
yarn build
```
